# SoulKnight
![image](https://user-images.githubusercontent.com/23046289/230697589-6940c4d6-b5f3-479d-a7b6-0637e0522f3e.png)

## 模型
- 3D
- 视角 45°

## 内容
- 背景
- 主角
- 子弹
- BOOS

## 逻辑
- BOOS 自动旋转寻找主角，找到后停止旋转开始攻击，然后重复
- 角色使用 WSAD 控制运动，可以使用鼠标左键发射子弹攻击 BOOS，击中后 BOOS 血条减一
- BOOS 血条减为零后死亡

## 技术
- 移动
- 弹幕
- 发射
- 碰撞
- 动画
- 状态机
- 对象池
